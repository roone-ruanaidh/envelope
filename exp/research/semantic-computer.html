Envelope · runtime sketch · unpromoted

The context window is a semantic cache, not the computer’s workspace.

The CPU runtime owns state, exact computation, memory, I/O, and acceptance. It issues bounded semantic instructions to an inference engine. The engine runs isolated programmatic sub-agents with temporary token views in GPU KV state. Tools remain I/O devices behind capability gates. Both return observations or receipts for the runtime to validate and store.
